# Todo App UI Aesthetics

16 variations of a simple todo app, each in a different visual style. Made by [Rasagy Sharma](https://www.linkedin.com/in/rasagy/) using [Codex](https://chatgpt.com/codex) (Free tier).

## Aesthetics

1. Simple & Minimal
2. Flat Design
3. Neumorphism
4. ASCII & Terminal inspired
5. Cyberpunk Dark Neon
6. Swiss Design
7. Skeuomorphism
8. Maximalism
9. Art Deco
10. Win95 UI
11. NES inspired Pixel UI
12. Futuristic
13. Aurora / Vista style
14. Material Design System
15. Biomorphic
16. Shadcn Design System
